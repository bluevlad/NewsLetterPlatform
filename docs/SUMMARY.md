# Summary

28년간 C++에서 Java, Spring Boot, React, AI/RAG까지
기술 변천을 직접 경험한 풀스택 시스템 전문가입니다.

현대자동차, 한화솔루션, KT&G, 청와대 등
대기업·공공기관 대상 30건 이상의 프로젝트를 수행했으며,

Documentum ECM 기반 엔터프라이즈 문서 관리 시스템을
4년 7개월간 운영·고도화한 ECM 전문성을 보유하고 있습니다.

교육 도메인에서 15년 이상 학원운영시스템, E-HRM,
해외교육관리 등을 설계·개발·운영한 경험이 있으며,

현재는 6개 마이크로서비스(www.unmong.com)를
Docker/CI-CD 기반으로 직접 설계하고 운영하고 있습니다.

AI 개발 도구를 활용한 1인 풀스택 개발 체계를 구축하여,
소규모 인력으로도 엔터프라이즈 수준의 시스템 개발이 가능합니다.

요구사항 분석부터 아키텍처 설계, 개발, 배포, 운영까지
전체 라이프사이클을 원스톱으로 수행할 수 있으며,

기술과 비즈니스 사이에서 기획자, 현업 담당자, 인프라 엔지니어 간의
소통을 중재하는 것이 가장 큰 강점입니다.
