## 2. 핵심 강점

### 28년 실무 경력 기반의 원스톱 수행 능력

1997년부터 현재까지 **C++ → ASP → Java → Spring Boot → React → AI/RAG**로 이어지는 기술 변천을 직접 경험하며, **요구사항 분석 → 아키텍처 설계 → 프로그램 개발 → 배포 → 운영**의 전체 소프트웨어 라이프사이클을 1인이 수행할 수 있는 역량을 보유하고 있습니다.

### 대기업·공공기관 프로젝트 수행 경험

현대자동차, 한화솔루션, KT&G, 현대·기아자동차, 청와대, 특수교육원, 대한보건산업협회 등 **대기업 및 공공기관 대상 30건 이상의 프로젝트**를 성공적으로 수행한 실적을 보유하고 있습니다.

### ECM 솔루션 전문성

Documentum ECM 기반으로 한화솔루션(3년 7개월), 현대자동차(1년)의 **엔터프라이즈 문서 관리 시스템 운영 및 고도화**를 수행하며, AWS/Docker 기반 인프라 설계부터 운영까지의 전문성을 축적했습니다.

### 교육 도메인 전문성

윌비스 학원 시스템 설계·개발·운영(7년), 특수교육원 원격연수원, 현대·기아자동차 해외교육관리시스템, KT&G E-HRM 등 **교육 시스템 전문 개발 경험 15년 이상**을 보유하고 있습니다.

### AI 활용 1인 개발 체계 구축

Claude Opus 4, Gemini 3 Pro 등 최신 AI 개발 도구를 활용한 **AI-Assisted 풀스택 개발 체계**를 자체 구축하여, 소규모 인력으로도 엔터프라이즈 수준의 시스템 개발이 가능합니다.

### 기술-비즈니스 간 소통 브릿지

단순 코딩이 아닌, 시스템 전체를 설계하고 직접 운영한 경험을 기반으로 **기획자, 현업 담당자, 인프라 엔지니어 간의 원활한 의사소통**을 중재할 수 있는 강점을 가지고 있습니다.

## 3. 운영 서비스 포트폴리오

현재 [www.unmong.com](http://www.unmong.com) 포털을 통해 아래 서비스들을 직접 설계하고 운영하고 있습니다.

### 인프라 운영 현황

| 항목 | 내용 |
|------|------|
| **서비스 포털** | www.unmong.com (HTTPS 지원) |
| **도메인 체계** | www / study / admin .unmong.com |
| **서비스 수** | 6개 마이크로서비스 (15+ 엔드포인트) |
| **컨테이너** | Docker Compose 기반 오케스트레이션 |
| **CI/CD** | GitHub Actions + Self-hosted Runner 자동 배포 |
| **운영 서버** | macOS (OrbStack/Docker) |

### 3-1. AllergyInsight - 알러지 진단 & 처방 플랫폼

AI 기반 알러지 검사 결과 분석 및 맞춤형 처방 정보 제공 시스템

| 기능 | 설명 |
|------|------|
| **RAG 기반 논문 검색/Q&A** | Retrieval-Augmented Generation 기술로 알러지 관련 논문 검색 및 질의응답 |
| **처방 대시보드** | 검사 결과 기반 지능형 처방 정보 시각화 |
| **뉴스레터** | 알러지 관련 최신 연구/뉴스 자동 큐레이션 및 발송 |
| **관리자 콘솔** | 사용자/데이터/콘텐츠 통합 관리 |

> **적용 기술**: Python, RAG, PostgreSQL, Docker

### 3-2. TeacherHub - 강사 분석 플랫폼

공무원 학원 강사 관련 카페/커뮤니티 게시글 동향 수집 및 분석 서비스

| 기능 | 설명 |
|------|------|
| **동향 분석 대시보드** | 강사별 커뮤니티 반응/트렌드 시각화 |
| **뉴스레터** | 분석 결과 정기 리포트 자동 발송 |

> **적용 기술**: Java Spring Boot, React, PostgreSQL, Docker

### 3-3. AcademyInsight - 학원 분석 플랫폼

학원 업계 동향 정보 수집 및 인사이트 제공 서비스

| 기능 | 설명 |
|------|------|
| **데이터 분석 대시보드** | 학원 관련 데이터 수집/분석/시각화 |
| **뉴스레터** | 학원 업계 동향 정기 리포트 자동 발송 |

> **적용 기술**: Java Spring Boot, React, PostgreSQL, Docker

### 3-4. HopenVision - 입시 예측 시스템

데이터 기반 입시 합격 예측 및 통계 분석 시스템

| 기능 | 설명 |
|------|------|
| **합격 예측** | 데이터 기반 입시 결과 예측 모델 |
| **통계 분석** | 입시 데이터 통계 시각화 |
| **관리자 콘솔** | 예측 모델 관리 및 데이터 운영 |

> **적용 기술**: Java Spring Boot, React, PostgreSQL, Docker

### 3-5. HealthPulse - 디지털 헬스케어 뉴스 서비스

디지털 헬스케어 뉴스 수집 및 리포트 자동화 서비스

| 기능 | 설명 |
|------|------|
| **뉴스 수집/큐레이션** | 건강/헬스케어 분야 뉴스 자동 수집 |
| **구독 관리** | 사용자 구독 신청 및 관리 |
| **관리자 콘솔** | 콘텐츠 관리 및 발송 이력 관리 |

> **적용 기술**: Python, PostgreSQL, Docker

### 3-6. Online Academy - 학원 관리 시스템

Java Spring Boot + React 기반 온라인 학원 관리 시스템

| 기능 | 설명 |
|------|------|
| **사용자 포털** | 수강생용 학습 관리 및 정보 조회 |
| **관리자 콘솔** | 회원/학사/회계/매출 통합 관리 |
| **REST API** | Swagger UI 기반 API 문서 자동화 |

> **적용 기술**: Java Spring Boot, React, PostgreSQL, Docker

## 6. 고객에게 제공할 수 있는 가치

### 시스템 설계 & 컨설팅

- 신규 시스템의 **아키텍처 설계 및 기술 스택 선정** 자문
- 기존 레거시 시스템의 **마이크로서비스 전환 전략** 수립 (ASP→Java 전환 실적 보유)
- Docker/클라우드 기반 **인프라 설계 및 마이그레이션** 컨설팅

### 풀스택 시스템 개발

- Backend(Java/Python) + Frontend(React) **풀스택 개발**
- REST API 설계 및 Swagger 기반 API 문서 자동화
- PostgreSQL/Oracle/MS-SQL 기반 데이터베이스 설계 및 최적화

### AI 기반 시스템 구축

- **RAG 기반 지능형 검색/Q&A 시스템** 구축 (AllergyInsight 실 운영 사례 보유)
- AI 기반 **데이터 수집/분석/뉴스레터 자동화** 파이프라인 구축
- LLM을 활용한 **업무 자동화 시스템** 개발

### ECM & 문서 관리

- Documentum ECM 기반 **엔터프라이즈 문서 관리 시스템** 구축/운영/고도화
- 검색엔진(SF-1), PDF 뷰어(Brava Viewer) 등 **솔루션 연동 개발**
- AWS/Docker 기반 ECM 인프라 설계

### DevOps & 운영

- Docker Compose 기반 **멀티 서비스 컨테이너 오케스트레이션**
- GitHub Actions 기반 **CI/CD 파이프라인** 구축
- 멀티 도메인 환경의 **서비스 포털 통합 관리** 체계 구축

### 도메인 전문성

- **교육 업계**: 학원 관리(회원/학사/회계), 강사 분석, 입시 예측, E-HRM, 해외교육관리 (15년+)
- **자동차**: 현대자동차 기술컨텐츠 플랫폼, 결제 차세대 시스템, 해외 딜러 교육관리
- **헬스케어**: AI 기반 알러지 진단/처방, 건강 뉴스 자동화 서비스
- **ECM**: Documentum 기반 문서 관리 시스템 운영/고도화 (한화솔루션, 현대자동차)
