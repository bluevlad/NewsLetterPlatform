뉴스레터 아카이브 가이드

발송된 뉴스레터를 언제든 다시 확인할 수 있는 아카이브 서비스

1. 서비스 소개

뉴스레터 아카이브는 발송 완료된 일일, 주간, 월간 뉴스레터를 웹에서 열람할 수 있는 서비스입니다. 구독자가 아니더라도 과거 뉴스레터를 확인하거나, 놓친 뉴스레터를 다시 읽을 수 있습니다.

대상 사용자

뉴스레터 구독자 일반 방문자

2. 주요 기능

📚

통합 아카이브

전체 테넌트(EduFit, AllergyInsight 등)의 뉴스레터를 한 화면에서 통합 조회합니다.

📅

월별 그룹핑

뉴스레터가 월별로 자동 분류되어 원하는 시기의 콘텐츠를 빠르게 찾을 수 있습니다.

🏷

유형별 구분

일일, 주간, 월간 뉴스레터가 색상 배지로 구분되어 한눈에 유형을 파악할 수 있습니다.

📨

원본 HTML 보기

실제 발송된 이메일과 동일한 HTML 원본을 브라우저에서 그대로 확인합니다.

3. 사용 방법

  1. 아카이브 페이지 접속
    통합 아카이브(/archive)에 접속하면 전체 테넌트의 뉴스레터가 표시됩니다. 특정 테넌트의 아카이브만 보려면 /{tenant_id}/archive로 접속합니다.
  2. 원하는 뉴스레터 선택
    월별로 그룹핑된 목록에서 날짜, 유형(일일/주간/월간), 제목을 확인하고 원하는 항목을 클릭합니다.
  3. 뉴스레터 열람
    선택한 뉴스레터의 HTML 원본이 브라우저에 렌더링됩니다. 실제 이메일과 동일한 내용을 확인할 수 있습니다.
  4. 목록으로 돌아가기
    브라우저의 뒤로가기 버튼으로 아카이브 목록으로 돌아갑니다.

4. 주의사항

참고

아카이브는 최근 발송된 뉴스레터부터 최대 100건(통합) 또는 50건(테넌트별)까지 표시됩니다.

참고

아카이브에서 열람하는 뉴스레터는 실제 발송된 이메일의 HTML 원본입니다. 이메일 클라이언트에 따라 실제 수신 시 보이는 화면과 약간 다를 수 있습니다.

5. 자주 묻는 질문

Q. 아카이브에서 뉴스레터가 보이지 않습니다
아카이브 기능이 활성화된 이후 발송된 뉴스레터만 표시됩니다. 이전에 발송된 뉴스레터는 포함되지 않을 수 있습니다.
Q. 통합 아카이브와 테넌트별 아카이브의 차이점은?
통합 아카이브(/archive)는 모든 서비스의 뉴스레터를 한번에 표시합니다. 테넌트별 아카이브(/{tenant}/archive)는 해당 서비스의 뉴스레터만 표시합니다.
Q. 일일, 주간, 월간 뉴스레터의 차이는?
일일(Daily)은 매일 발송되는 최신 소식, 주간(Weekly)은 한 주간의 요약, 월간(Monthly)은 한 달간의 종합 분석 뉴스레터입니다.
Q. 아카이브 뉴스레터를 다른 사람과 공유할 수 있나요?
네, 뉴스레터 상세 페이지의 URL을 그대로 공유하면 됩니다. 별도 로그인 없이 누구나 열람 가능합니다.
Q. 아카이브에서 구독 신청도 가능한가요?
아카이브 목록 하단의 "구독 페이지로 돌아가기" 링크를 클릭하거나, 테넌트별 아카이브에서 해당 서비스의 구독 페이지로 이동할 수 있습니다.
